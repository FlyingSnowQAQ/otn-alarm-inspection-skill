# OTN 告警巡检工具 — 快速入门

## 环境要求

- Node.js 18+
- npm 9+
- 浏览器：Chrome 90+ / Edge 90+

## 一键启动

```bash
cd otn-alarm-tool
npm install          # 首次使用：安装依赖
npm run dev          # 启动开发服务器 → http://localhost:5173
```

## 生产构建

```bash
npm run build
# 输出：dist/index.html（单文件，可直接双击打开）
```

## 使用流程

### 1. 导入数据

1. 打开工具，进入"数据导入"页面
2. 顶部 Chip 选择导入模式：
   - `导入告警数据`：拖入烽火告警查询报表 .xls/.xlsx（26个字段自动映射）
   - `导入网元信息`：拖入烽火网元查询报表 .xls/.xlsx（18个字段自动映射）
3. 检查自动列映射，点击"开始导入"
4. 导入完成后，数据自动持久化（刷新不丢失）
5. 支持**清除网元数据**（含省份统计展示）
6. 支持**追加导入**（同一项目内多次导入数据自动合并）

### 2. 查看拓扑

1. 切换到"拓扑视图"页面
2. 有网元的省份自动高亮，显示网元数和告警统计
3. 点击活跃省份 → 进入省份内站点详情
4. 点击灰色省份 → 无响应（尚未导入数据）

### 3. 告警分析

1. 切换到"告警分析"页面
2. 默认明细视图：多维筛选（级别/类型/省份/状态/时间）
3. 切换到"聚合视图"：按时间窗口聚合告警，自动识别根因
4. 调整时间窗口（5/15/30/60分钟）观察聚合粒度

### 4. 仪表盘

- 总览统计卡片（紧急/主要/次要/提示）
- 告警趋势图（按日堆叠面积图）
- 告警类型分布饼图
- 告警风暴 Top10 卡片
- 高频闪告单盘 Top10 卡片
- 最近告警列表 + 网元健康度排名 + 省份统计

### 5. 报表

- 查看告警汇总/网元巡检报表
- 一键导出为 Excel

### 6. 项目管理

- 新建项目/导入项目来管理多套数据集
- 导出 JSON 备份
- 项目间切换数据自动恢复
